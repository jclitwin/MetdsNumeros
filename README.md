# Ferramenta para Métodos Númericos

Ferramenta ainda em desenvolvimento, tais como foram feitos poucos testes para cálculos e confirmação se realmente está 100% funcional.

### Features
- Derivada Númerica
- Bissecção
- Posição Falsa
- Posição Ponto Fixo
- Newton-Raphson
- Secantes *
